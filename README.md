# Directory Sync Tool

## What this project does
A Python tool that analyzes and compares two directories to identify differences between them.

## Features implemented
- Recursively scan directories
- Collect file metadata (name, extension, size)
- Detect new files
- Detect modified files (based on size)
- Detect missing files

## Features in progress
- Copy new/missing files to destination
- Delete extra files from destination
- Logging system for tracking changes
- CLI arguments for better control
